This add-on allows to scan a given URL with multiple virus scanners. VirusTotal provides this service. See https://www.virustotal.com/. Right click on the URL. Select VirusTotal Scan. Observe the scan results. By default the scan results are displayed in a new window. A new tab display configuration option is available. This add-on requests the following permissions.
  • Access your data for virustotal.com in order to perform the scan.
  • Display notifications to you in order to display progress or errors.
For issues such as bugs, feature requests, questions, problem support, or suggestions use https://github.com/2000me/vtscan/issues. Contributions such as localizations are welcome as pull requests. https://github.com/2000me/vtscan/pulls.
